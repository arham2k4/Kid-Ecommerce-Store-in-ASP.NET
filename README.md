# 🛒 E-Commerce MVC Web Application

A modern e-commerce web application built using ASP.NET MVC and C#. The application provides product browsing, shopping cart management, order processing, and an intuitive user interface following the Model-View-Controller (MVC) architecture.

## 🚀 Features

* 🏠 Home Page with Product Listings
* 📦 Product Catalog and Details
* 🛒 Shopping Cart Functionality
* 📋 Order Checkout and Confirmation
* 🗂️ Category Management
* ⚙️ Admin Panel for Product Management
* ✏️ Create, Update, and Delete Products
* 🎨 Responsive User Interface
* 🏗️ MVC Architecture

## 🛠️ Technologies Used

* ASP.NET MVC
* C#
* Entity Framework
* SQL Server
* HTML5
* CSS3
* Bootstrap
* Razor Views

## 📌 Modules

### 🏠 Home Module

Displays products available in the store.

### 📦 Product Module

View product details and browse products.

### 🛒 Cart Module

Add, remove, and manage shopping cart items.

### 📋 Order Module

Checkout and confirm customer orders.

### ⚙️ Admin Module

Manage products through create, edit, and delete operations.

## ▶️ Getting Started

1. Clone the repository:

```bash id="a6lrnh"
git clone https://github.com/your-username/ECommerceMVC.git
```

2. Open the solution in Visual Studio.

3. Restore NuGet packages.

4. Configure the database connection string in `appsettings.json`.

5. Run the application.
